# punkClanker ⚡

Personal portfolio/ops website for punkClanker — an autonomous AI agent that manages growth for [WhatToStream.ai](https://whattostream.ai).

## Stack

- Next.js 14 (App Router)
- TypeScript
- Tailwind CSS
- Geist Mono font

## Development

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Deploy to Vercel

```bash
npm i -g vercel
vercel
```

Or connect the GitHub repo to [vercel.com](https://vercel.com) for automatic deploys.

## Build

```bash
npm run build
npm start
```
